API 使用指南


                    
                    

                        
API 端点

                        

                            
GeminiCLI

                            

                                POST
                                /v1/chat/completions
                            

                            

                                POST
                                /v1/messages
                            

                            

                                POST
                                /v1/models/{model}:generateContent
                            

                            

                                POST
                                /v1/models/{model}:streamGenerateContent
                            

                            

                                GET
                                /v1/models
                            

                        

                        

                            
Antigravity

                            

                                POST
                                /antigravity/v1/chat/completions
                            

                            

                                POST
                                /antigravity/v1/messages
                            

                            

                                POST
                                /antigravity/v1/models/{model}:generateContent
                            

                            

                                POST
                                /antigravity/v1/models/{model}:streamGenerateContent
                            

                            

                                GET
                                /antigravity/v1/models
                            

                        

                        

                            
Codex

                            
通过 OpenAI Codex API 转发，需 OpenAI 凭证

                            

                                POST
                                /codex/v1/chat/completions
                            

                            

                                POST
                                /codex/v1/messages
                            

                            

                                GET
                                /codex/v1/models
                            

                        

                    


                    
                    

                        
认证方式

                        

                            
支持以下任意方式，密码默认 pwd

                            

                                

                                    Header
                                    Authorization: Bearer pwd
                                

                                

                                    Header
                                    x-goog-api-key: pwd
                                

                                

                                    Header
                                    x-api-key: pwd
                                

                                

                                    Param
                                    ?key=pwd
                                

                            

                        

                    


                    
                    

                        
支持的模型

                        

                            
GeminiCLI 基础模型

                            
每个凭证 1000 次请求，1M 上下文

                            

                                gemini-2.5-pro
                                gemini-2.5-flash
                                gemini-3-pro-preview
                                gemini-3-flash-preview
                                gemini-3.1-pro-preview
                            

                        

                        

                            
Antigravity 模型

                            
从 API 动态获取，包含 Claude 和 Gemini 系列

                            

                                claude-sonnet-4-5
                                claude-opus-4-5
                                claude-sonnet-4-6
                                gemini-2.5-flash
                                gemini-2.5-pro
                                ...
                            

                            
以 /antigravity/v1/models 返回结果为准

                        

                        

                            
Codex 模型

                            
GPT-5、o 系列等 OpenAI 模型

                            

                                gpt-5
                                gpt-5-codex
                                gpt-5.1
                                gpt-5.2
                                gpt-5.4
                            

                            

                                gpt-4.1-nano
                            

                            
支持思考等级后缀，如 gpt-5(high)、gpt-5.4(medium)

                        

                    


                    
                    

                        
模型后缀与前缀

                        

                            
思考模式后缀

                            
控制推理预算（GeminiCLI）

                            
2.5 系列

                            

                                -max
                                -high
                                -medium
                                -low
                                -minimal
                            

                            
3 Flash

                            

                                -high
                                -medium
                                -low
                                -minimal
                            

                            
3 Pro

                            

                                -high
                                -low
                            

                        

                        

                            
思考等级（Codex）

                            
模型名后加 (level) 控制推理深度

                            

                                none
                                minimal
                                low
                                medium
                                high
                                xhigh
                            

                            
示例：gpt-5(high)、gpt-5.4(medium)、gpt-5.4(xhigh)

                        

                        

                            
其他后缀

                            
-search 搜索增强，可与思考后缀组合

                        

                        

                            
功能前缀

                            

                                

                                    假流式/
                                    非流式转 SSE
                                

                                

                                    流式抗截断/
                                    自动续写
                                

                            

                        

                        

                            
图像后缀（AG）

                            
模型：gemini-3-pro-image

                            
分辨率

                            

                                -2k
                                -4k
                            

                            
比例

                            

                                -1x1
                                -16x9
                                -9x16
                                -21x9
                                -4x3
                                -3x4
                            

                            
组合示例：gemini-3-pro-image-4k-16x9

常见错误

                        

                            

                                429
                                No capacity available for model gemini-** on the server
                            

                            
Google 服务端容量不足，非凭证问题。高峰时段出现，等待后重试即可，系统会自动重试。

                        

                        

                            

                                403
                                This service has been disabled in this account for violation of Terms of Service
                            

                            
账号被 Google 取消 Gemini CLI / Antigravity 权限，凭证永久失效，需移除并更换账号。

                        

                        

                            

                                403
                                VALIDATION_REQUIRED — To continue, verify your account ...
                            

                            
临时风控，需登录对应账号访问验证链接完成手机号验证即可解除，验证后凭证可继续使用。

                        

                        

                            

                                400/500
                                Bad Request
                            

                            
一般代表 Google GCP 权限被封禁，凭证失效，需移除并更换账号。
