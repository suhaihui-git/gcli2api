                                Claude 格式
                            

                            

                                POST
                                /antigravity/v1/models/{model}:generateContent
                                Gemini 非流式
                            

                            

                                POST
                                /antigravity/v1/models/{model}:streamGenerateContent

认证方式

                        

                            
所有端点均需认证，支持以下任意方式（密码默认为 pwd，可通过环境变量修改）：

                            

                                

                                    Header
                                    Authorization: Bearer your_password
                                

                                

                                    Header
                                    x-goog-api-key: your_password
                                

                                

                                    Header
                                    x-api-key: your_password
                                    Claude 格式专用
                                

                                

                                    Param
                                    ?key=your_password
                                    URL 参数
                                

                            

                        

                    


                    
                    

                        
支持的模型

                        

                            
GeminiCLI 基础模型

                            
每个凭证提供 1000 次请求额度，所有模型均支持 1M 上下文窗口

                            

                                gemini-2.5-pro
                                gemini-2.5-flash
                                gemini-3-pro-preview
                                gemini-3-flash-preview
                                gemini-3.1-pro-preview
                            

                        

                        

                            
Antigravity 模型

                            
AG 模型从 Google API 动态获取，通常包括 Claude 和 Gemini 系列

                            

                                claude-sonnet-4-5
                                claude-opus-4-5
                                claude-sonnet-4-6
                                claude-sonnet-4-6-thinking
                                gemini-2.5-flash
                                gemini-2.5-pro
                                ...
                            

                            
AG 可用模型以 /antigravity/v1/models 返回结果为准

                        

                    


                    
                    

                        
模型后缀与前缀

                        

                            
思考模式后缀（GeminiCLI）

                            
在基础模型名后添加后缀来控制推理预算

                            
Gemini 2.5 系列

                            

                                -max
                                -high
                                -medium
                                -low
                                -minimal
                            

                            
Gemini 3 Flash 系列

                            

                                -high
                                -medium
                                -low
                                -minimal
                            

                            
Gemini 3 Pro 系列

                            

                                -high
                                -low
                            

                            
示例：gemini-2.5-pro-max、gemini-2.5-flash-high

                        

                        

                            
搜索后缀

                            
在模型名后添加 -search 启用搜索增强

                            
可与思考后缀组合：gemini-2.5-pro-high-search

                        

                        

                            
功能前缀

                            

                                

                                    假流式/
                                    非流式响应转换为 SSE 流式格式，用于兼容仅支持流式的客户端
                                

                                

                                    流式抗截断/
                                    自动检测响应截断并续写，确保获得完整回答
                                

                            

                            
示例：假流式/gemini-2.5-pro、流式抗截断/gemini-2.5-pro-max

                        

                        

                            
图像生成后缀（Antigravity）

                            
基础模型：gemini-3-pro-image

                            
分辨率

                            

                                -2k
                                -4k
                            

                            
比例

                            

                                -1x1
                                -16x9
                                -9x16
                                -21x9
                                -4x3
                                -3x4
                            

                            
可组合使用：gemini-3-pro-image-4k-16x9（4K 横屏）

                        

                    


                    
                    

                        
请求示例

                        

                            
                            
                            
                        

                        

                            

                                
端点：/v1/chat/completions 或 /antigravity/v1/chat/completions

                                
{
  "model": "gemini-2.5-pro",
  "messages": [
    {"role": "system", "content": "You are a helpful assistant"},
    {"role": "user", "content": "Hello"}
  ],
  "temperature": 0.7,
  "stream": true
}

                                
多模态请求

                                
{
  "model": "gemini-2.5-pro",
  "messages": [
    {
      "role": "user",
      "content": [
        {"type": "text", "text": "描述这张图片"},
        {"type": "image_url", "image_url": {"url": "data:image/jpeg;base64,..."}}
      ]
    }
  ]
}

                            

                            

                                
端点：/v1/models/{model}:generateContent 或 :streamGenerateContent

                                
{
  "contents": [
    {"role": "user", "parts": [{"text": "Hello"}]}
  ],
  "systemInstruction": {
    "parts": [{"text": "You are a helpful assistant"}]
  },
  "generationConfig": {
    "temperature": 0.7
  }
}

                            

                            

                                
端点：/v1/messages 或 /antigravity/v1/messages

                                
需要 Header：anthropic-version: 2023-06-01

                                
{
  "model": "gemini-2.5-pro",
  "max_tokens": 1024,
  "system": "You are a helpful assistant",
  "messages": [
    {"role": "user", "content": "Hello"}
  ]
}

{
  "choices": [{
    "message": {
      "role": "assistant",
      "content": "最终答案",
      "reasoning_content": "详细的思考过程..."
    }
  }]
}

                        

                    


                    
                    

                        
常见错误

                        

                            

                                429
                                No capacity available for model gemini-** on the server
                            

                            
Google 服务端容量不足导致的临时限流，非凭证问题。通常在高峰时段出现，等待后重试即可。系统已配置自动重试机制，一般无需手动处理。

                        

                        

                            

                                403
                                This service has been disabled in this account for violation of Terms of Service
                            

                            
该账号已被 Google 取消 Gemini CLI / Antigravity 使用权限，凭证永久失效。需移除该凭证并更换其他账号。

                        

                        

                            

                                403
                                PERMISSION_DENIED / VALIDATION_REQUIRED — To continue, verify your account at ...
                            

                            
账号触发 Google 临时风控，需要验证身份后恢复。在浏览器中登录对应 Google 账号，访问错误信息中的验证链接完成手机号验证即可解除。验证后凭证可继续使用。

                        

                        

                            

                                400/500
                                Bad Request
                            

                            
一般代表该账号的 Google GCP 权限已被封禁，凭证失效。需移除该凭证并更换其他账号。

                        

                    

                


                
                

                    

                        
项目信息

                        
项目信息详情

                    


                    

                        
Gemini API Pool

                        
一款将Google Gemini API转换为OpenAI和Gemini API格式的代理工具。支持多账号管理、自动轮换、实时日志监控等功能。

                        

                            仅供学习研究使用
                        

                    


                    

                        
功能特性

                        

                            
多账号管理：批量上传和管理Google账号

                            
自动轮换：智能凭证轮换以避免速率限制

                            
监控面板：使用统计、错误追踪、实时日志

                            
安全保护：OAuth2认证、异常账号自动封禁

                            
灵活配置：热更新设置、代理支持

                            
响应式UI：适配桌面端和移动端
